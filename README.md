# Como clonar o projeto

1. Abrir o Github;
2. Acessar repositorio
3. Copiar link
4. Abrir terminal (CMD, Powershell, Git Bash) na pasta desejada
5. Digitar o comando "git clone [Link repositorio]"
6. Digitar no terminal "code [nome da pasta]"


# Como adicionar o projeto no Github

1. git status
2. git add .
3. git status
4. git commit -m "[nome do projeto]"
5. git push
